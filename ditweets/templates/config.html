Pour révoquer les accès a votre compte twitter, il vous suffit de supprimer l'application twitter via https://apps.twitter.com.

Génération des clés Twitter

Pour obtenir les clés il faut créer une application Twitter.

Création d'une application Twitter

  • Aller sur la page https://apps.twitter.com.
  • Cliquer sur Create New App
  • Renseigner les champs :
    • Name: vous pouvez mettre ce que vous voulez.
    • Description: n'importe quoi de plus de 10 caractères.
    • Website: n'importe quoi sous la forme d'une adresse HTTP valide.
    • Callback URL: laisser vide.
    • Cocher le Developer Agreement.
  • Cliquer sur Create your Twitter application.
  • Cliquer sur l'onglet Keys and Access Tokens
  • Recopier les chaines Consumer Key et Consumer Secret
  • Cliquer sur le bouton Create my access Token
  • Recopier les chaines Access Token et Access Token Secret
